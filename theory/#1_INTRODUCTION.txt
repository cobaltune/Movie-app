#1.2 Why React

신기술을 배울 때 중요한 것
- 누가 이 기술을 사용하는지
- 그들의 규모가 얼마나 큰지
- 이 기술이 그들에게 얼마나 중요한지

1. 전 세계 상위 1만 개의 제일 큰 웹사이트 중 44.76%가 React JS를 사용하고 있음
   ex) 에어비앤비, 인스타그램, 페이스북, 넷플릭스
    

2. 페이스북이 React JS를 만들었고, 이걸로 페이스북 웹사이트 전부를 다시 만듦
   그리고 오늘날에도 페이스북은 React JS를 지원하고 있음
   (= React JS로 작업하고, 이를 개선하기 위해 자금을 투자하고 있으니 React JS는 폐기되지 않는다)
   +) 우리는 그 코드를 무료로 사용


3. React JS는 정말 큰 커뮤니티를 갖고 있음
   : React JS는 javascript와 매우 가깝기 때문, 실상 javascript의 커뮤니티를 거의 다 빌려온 셈인데 이 커뮤니티가 거대함 
     ⇒ 기술을 배울 때, 그 기술에 대한 생태계가 존재한다는 건 좋은 것
    
    ex 1. React JS를 기반으로 제작된 framework가 있음
    ex 2. React Native는 도구로, iOS or Android 어플을 React JS 코드로 만들 수 있게 해줌
    

total. 한번 배우면 어디든 쓸 수 있음
